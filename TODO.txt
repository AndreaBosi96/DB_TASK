To run the solution locally:
- import the db in your sql server (you will find it in the mail as requested)
- update the db connection string in appsettings.json according to your own system settings
- install necessary packages:
        -- Microsoft.Extensions.Configuration
        -- Microsoft.Extensions.Configuration.json
        -- System.Data.SqlClient
- get into the project folder and run simple dotnet cmds
        -- dotnet restore
        -- dotnet run
